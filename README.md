# react-native-apple-wallet

Apple wallet integration with add card to wallet

## Installation

```sh
npm install react-native-apple-wallet
```

## Usage

```js
import { multiply } from "react-native-apple-wallet";

// ...

const result = await multiply(3, 7);
```
